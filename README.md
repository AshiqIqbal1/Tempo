<h1 align="center">Tempo</h1>

<p align="center">
  A self-hosted music player for your own library.<br>
  Go backend · Flutter frontend · Docker deployment
</p>

---

I built Tempo because I wanted something simple that plays my own music files without subscriptions, tracking, or ads. Drop your mp3s on a server, point Tempo at the folder, and you've got a full music player on your phone.

## How it works

The backend scans your music folder, pulls out metadata and album art using taglib + ffprobe, and stores everything in SQLite. It serves tracks over a simple REST API. The Flutter app connects to it and handles playback — background audio, notification controls, the whole thing.

If your phone and server aren't on the same network (or your WiFi has AP isolation), Tailscale works perfectly.

## Features

- Background playback with notification + lock screen controls
- Full-screen player with blurred album art and waveform visualizer
- Mini player across all screens
- Search (titles, artists, albums)
- Playlists — auto-generated by decade + custom
- Artist and album browsing
- 11-preset equalizer (Rock, Pop, Jazz, etc.)
- Server-side shuffle — not limited to loaded tracks
- Today's Mixes — quick random queues (30/50/100 tracks)
- Queue view with swipe-to-remove
- Add to playlist via long-press
- Landscape support
- Manual library rescan from Settings

## Quick start

### Backend

```bash
# edit docker-compose.yml — set your music path
docker compose up -d --build
```

```yaml
# docker-compose.yml
volumes:
  - /your/music/folder:/music:ro
```

Server starts on `:8081`. Scans in background — API available immediately.

**Supported formats:** mp3, flac, m4a, m4b, ogg, opus

### Frontend

Set your server IP:

```dart
// frontend/lib/src/models/track.dart
const _baseUrl = 'http://YOUR_IP:8081';
```

Build and install:

```bash
cd frontend
flutter build apk --release
adb install build/app/outputs/flutter-apk/app-release.apk
```

## API

| Method | Endpoint | What |
|--------|----------|------|
| GET | `/tracks?limit=20&offset=0` | Paginated track list |
| GET | `/tracks/shuffle?limit=50` | Random tracks |
| GET | `/tracks/search?q=love` | Search |
| GET | `/tracks/{id}/stream` | Audio stream |
| GET | `/tracks/{id}/art` | Full album art |
| GET | `/tracks/{id}/art/thumbnail` | 256x256 thumbnail |
| GET | `/artists` | Artist list (5+ tracks) |
| GET | `/albums` | Album list with cover art |
| GET | `/playlists` | All playlists |
| POST | `/playlists` | Create playlist |
| POST | `/scan` | Trigger rescan |

## Network

If your router has AP isolation (most do by default), your phone can't reach the server over WiFi. Two options:

1. **Tailscale** — install on both, use Tailscale IP
2. **Disable AP isolation** — router-dependent

The app includes `network_security_config.xml` for cleartext HTTP on Android.
